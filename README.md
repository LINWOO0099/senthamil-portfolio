# Senthamil R - Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

✨ **Modern Design**
- Dark theme with blue accent colors
- Responsive layout that works on all devices
- Smooth animations and transitions
- Professional UI/UX

👨‍💼 **Sections**
- **Home**: Hero section with call-to-action buttons
- **About**: Personal bio and statistics
- **Skills**: Technical skills organized by category
- **Projects**: Featured project showcase
- **Contact**: Contact form and social links

⚡ **Interactive Features**
- Smooth scrolling navigation
- Form validation
- Scroll-to-top button
- Scroll animations
- Responsive hamburger menu
- Hover effects on cards

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, Grid, and Flexbox
- **JavaScript**: Interactivity and smooth scrolling
- **Responsive Design**: Mobile-first approach

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content with your own information

## Customization

### Update Personal Information

Edit `index.html` to update:
- Your name and professional title
- About section text
- Skills and projects
- Contact email and social links

### Change Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;      /* Main blue */
    --secondary-color: #1e40af;    /* Darker blue */
    --dark-bg: #0f172a;            /* Dark background */
    --accent-color: #3b82f6;       /* Accent blue */
}
```

### Add Projects

Duplicate a project card in the Projects section and update:
- Project title
- Description
- Technologies used
- Link to project

## Contact Form

The contact form currently validates input and shows a success message. To enable email functionality, you'll need to:

1. Set up a backend service (Node.js, Python, etc.)
2. Use a service like FormSubmit or Netlify Forms
3. Update the form submission handler in `script.js`

## Deployment

### Deploy on GitHub Pages

1. Push your repository to GitHub
2. Go to repository Settings
3. Enable GitHub Pages
4. Select `main` branch as source
5. Your site will be available at `https://LINWOO0099.github.io/senthamil-portfolio/`

### Deploy on Other Platforms

- **Netlify**: Connect your GitHub repo directly
- **Vercel**: Import your repository
- **Any hosting service**: Upload `index.html`, `styles.css`, and `script.js`
